# image_collector
__両方ともGoogleからの検索になります。__

# How to GG_Collector
例:
```
python GG_collector.py -t baseball -n 100 -d ./data
```   
-t: 集めたい画像のキーワード   
-n: 集めたい画像の枚数   
-d: 出力先のディレクトリパス   
-f: 出力先を上書きしていくか   　　　


# How to iCrawler
例:
```
python iCrawler.py --key baseball --dir ./data --N 100
```   
--key: 集めたい画像のキーワード   
--dir: 出力先のディレクトリパス   
--N: 集めたい画像の枚数
